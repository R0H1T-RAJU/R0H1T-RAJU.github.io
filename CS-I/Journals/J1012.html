1.1: Describe the general process for executing addition, regardless of base.
First align the digits vertically, then add each column starting from the right. If the sum of the column is 2 digits carry the tens place digit to the next column and repeat the process for all the columns.

2.1: What special factors need to be considered when executing addition with octal numbers?
Add each digit as usual, but if the sum is larger than 7 add 2 in order to convert it into base 8.

3.1: What special factors need to be considered when executing addition with hexadecimal numbers?
On the last column, if the sum is greater than 9 and less than 16 use letters to represent the sum (A-F).

4.1: What special factors need to be considered when executing addition with binary numbers?
If the sum of a column is 2 or 3, the binary number is 10 and 11 respectively, and you must carry the one.
